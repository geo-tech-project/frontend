MapComponent

MapComponent()

This class is made to handle the map/input page of the application. It contains the map and the form with the input fields. It also contains the uploader for the model/training data file. If the user submitted the form, the data is sent to the server. Condional to the servers response, the class either redirect to the result page or displays an error message via bulma-toast.
