MapComponent

MapComponent(fb, http, router)

This class is made to handle the map/input page of the application. It contains the map and the form with the input fields. It also contains the uploader for the model/training data file. If the user submitted the form, the data is sent to the server. Condional to the servers response, the class either redirect to the result page or displays an error message via bulma-toast.

# initMap() → {void}

Initializes the map. Set the center of the map to the center of Münster. Set the zoom level to 12. Setting the zoom control to the top right corner. Also adding the tile Layer from the OpenStreetMap. Adding the FeatureGroup for displaying the training data polygons. Also adding the Draw control to the map, which is used for drawing the AOI. Controlling the storing of the drawn data, so that the user can only draw one rectangle.

# ngOnInit() → {void}

After the page is loaded, this function is called. It initializes the map and the form. It is setting the validators for the form. Also handling the functionality, that if training data are uploaded, the polygons are displayed on the map.
